HR Attrition Analysis Dashboard

Overview
This project focuses on analysing employee attrition data to identify workforce trends and key factors influencing employee turnover. The goal is to transform raw HR data into meaningful insights through structured analysis and interactive visualisation, helping organisations make data-driven HR decisions.


  Objectives
- Analyse employee attrition patterns
- Identify high-risk departments and roles
- Study the impact of salary and experience on retention
- Build interactive dashboards for business insights
- Present clear and actionable workforce analytics



  Tools & Technologies
- SQL –> Data cleaning, transformation, and querying
- Power BI –> Dashboard design and KPI visualisation
- Python (Basic) –> Data exploration and preprocessing
- Excel / CSV –> Data validation and preparation



   Project Workflow
1. Data Preparation
   - Removed duplicates and handled missing values
   - Standardised data formats
   - Structured datasets for analysis

2. Data Analysis
   - Department-wise attrition trends
   - Salary vs turnover analysis
   - Experience level and workforce insights

3. Dashboard Development
   - KPI metrics for attrition rate
   - Interactive filters and slicers
   - Visual reports for HR decision-making



  Key Insights
- Certain departments showed higher attrition levels
- Salary ranges influenced employee retention
- Early-career employees had higher turnover patterns
- Workforce data revealed potential retention risks



Business Value
This project demonstrates how data analytics can help organisations:
- Improve workforce planning
- Identify employee retention issues
- Make informed HR decisions through data storytelling
